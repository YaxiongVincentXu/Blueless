# Bluelesswork

Bluelesswork documents a practical remediation plan for replacing hard-to-read bright-light-blue critical characters in a third-party database front end.

## Contents

- [`docs/blue-contrast-remediation.md`](docs/blue-contrast-remediation.md) describes the color-change policy, rollout checklist, and acceptance criteria.
- [`config/accessible-blue-palette.json`](config/accessible-blue-palette.json) records replacement-color options and rollout requirements in a machine-readable form.
